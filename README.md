# NotCo Colombia — Proyecto Web

Sitio web de práctica inspirado en la marca NotCo, desarrollado como proyecto para el programa SENA.

🔗 **Sitio en vivo:** https://laurarojaspardo-dot.github.io/NotCo-Project/

## Sobre el proyecto

Este proyecto recrea el sitio web de NotCo Colombia con fines educativos, aplicando conceptos de maquetación, diseño responsivo e integración de contenido multimedia.

## Tecnologías usadas

- HTML5
- CSS3
- JavaScript

## Aviso

Este es un proyecto educativo/de práctica sin fines comerciales, realizado como parte de un programa de formación. No está afiliado, respaldado ni asociado oficialmente con NotCo. Todas las marcas, nombres e imágenes pertenecen a sus respectivos dueños.
